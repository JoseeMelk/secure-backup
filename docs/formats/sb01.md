# SB01 Format Specification

Status: Legacy

---

## Layout

| Offset | Size | Field |
|---|---|---|
| 0 | 4 | MAGIC (`SB01`) |
| 4 | 16 | Salt |
| 20 | 12 | Nonce |
| 32 | 16 | Authentication Tag |
| 48 | N | Ciphertext |

---

## Limitations

- No metadata support
- Entire file may be processed in memory
- Not optimized for streaming
